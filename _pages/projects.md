## ScholEval-AgentBench
**Benchmark for locally-run LLM agents on Chinese scholarly evaluation tasks**

A benchmark designed to evaluate how locally-run LLM-powered agents perform on Chinese scholarly evaluation tasks, with focus on **value alignment** between agent reasoning and human expert judgment.

- **Task families** (v0): paper summarization, claim-evidence extraction, quality dimension scoring, consistency checking
- **Setup**: Single-agent + tool use, local environment (LM Studio compatible)
- **Privacy-aware**: All public dev samples are de-identified; PKU research data excluded from public release
- **Repository**: [github.com/fightforlearn/ScholEval-AgentBench](https://github.com/fightforlearn/ScholEval-AgentBench)
- **Status**: v0 released 2026-05; v1 (planner + worker dual-agent) planned for 2026 H2

## PhD Thesis (in progress)
**AI value alignment in academic evaluation: A policy-mechanism study based on single-expert simulation**

(中文：AI 时代学术评价中的价值观对齐：基于单专家模拟的政策机制研究)

- Combines expert interviews, functional framework modeling, and LLM agent simulation
- Investigates how multiple expert reasoning chains diverge in evaluating the same scholarly work
- Implications for policy design in AI-assisted scholarly evaluation
- Expected completion: 2027 Q1
